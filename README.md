# Date Parsing

Easy Python function to parse many different date formats...

    "11-10-1975"    1975-10-11 00:00:00
    "11/10/75"      1975-10-11 00:00:00
    "11.10.75"      1975-10-11 00:00:00
    "11 10 75"      1975-10-11 00:00:00
    "11/10"         2014-10-11 00:00:00
    "11-10"         2014-10-11 00:00:00
    "11.10"         2014-10-11 00:00:00
    "11 10"         2014-10-11 00:00:00
    "11"            2014-07-11 00:00:00
    "ciao"          2014-07-18 16:48:32.803871
